# Project Overview

## 项目名称

BOSS 岗位智能筛选助手

## 项目定位

这是一个面向 BOSS 直聘岗位列表页的 Chrome Extension，用于辅助求职者根据岗位偏好和简历内容筛选岗位、查看匹配原因，并生成后续沟通所需的招呼语。

## 核心目标

- 在 BOSS 岗位列表页同步职位名、工作地、经验、学历和薪资筛选条件。
- 滚动抓取岗位列表，并按设定数量收集岗位。
- 根据用户简历、岗位信息和求职偏好进行规则评分或 AI 评分。
- 在插件面板中集中展示匹配岗位，支持人工审核和勾选。
- 对勾选岗位生成打招呼队列，并打开对应 JD 页面供用户继续确认。

## 当前能力

- 支持 OpenAI 和 DeepSeek API 配置。
- 支持 API 连接测试。
- 支持本地规则初筛和 AI 深度匹配。
- 支持审核结果、勾选状态和打招呼队列本地持久化。
- 支持后台新标签打开 JD，避免覆盖原始 BOSS 列表页。

## 当前边界

- 不自动替用户点击 BOSS 的最终发送动作。
- 不保证适配 BOSS 未来所有页面结构变化。
- 不解析 PDF、Word 等简历文件，当前使用用户粘贴的简历文本。
- AI 判断质量依赖用户提供的简历内容、JD 文本完整度和模型返回质量。

## 数据与隐私

- API Key 存储在本机 Chrome `storage.local`。
- 启用 AI 判断时，简历内容、岗位信息和偏好会发送给用户配置的 AI 服务商。
- 请不要将真实 API Key、简历全文或个人敏感信息提交到代码仓库。
