# User Guide

## 安装

1. 打开 Chrome，进入 `chrome://extensions/`。
2. 打开“开发者模式”。
3. 点击“加载已解压的扩展程序”。
4. 选择本项目根目录，也就是包含 `manifest.json` 的文件夹。
5. 打开 BOSS 直聘岗位列表页。
6. 点击浏览器工具栏中的插件图标。

## 使用流程

1. 在插件中填写 AI 服务商、模型和 API Key。
2. 点击“测试连接”，连接成功后会自动启用 AI 判断。
3. 填写筛选条件：职位名、工作地、工作经验、学历要求、薪资范围、排除词和简历内容。
4. 插件会尝试将筛选条件同步到 BOSS 页面。
5. 点击“抓取并评分”。
6. 在“审核确认”中查看岗位匹配分、通过原因或不通过原因。
7. 勾选希望继续沟通的岗位。
8. 点击“一键打招呼”生成打招呼队列，并批量打开 JD 页面。
9. 在 BOSS 页面中人工确认 JD，并手动进入沟通流程。

## 打招呼流程

当前版本不会自动发送消息。推荐流程：

1. 生成招呼语。
2. 打开岗位 JD。
3. 手动点击“立即沟通”进入聊天页。
4. 先发送简历或作品图片。
5. 复制并发送招呼语。
6. 回到插件中标记成功、跳过或失败。

## 常见问题

### API 连接成功，但运行时报 Receiving end does not exist

通常是扩展重新加载后，原来的 BOSS 标签页没有重新注入 content script。请刷新 BOSS 页面后再运行。

### 筛选条件没有同步到 BOSS

BOSS 页面结构可能变化，或当前页面不是岗位列表页。请先确认当前打开的是 BOSS 岗位列表页，再重新选择筛选项。

### 审核结果打开新 tab 后消失

当前版本会将上次审核结果保存到本机 Chrome storage。重新打开插件会恢复上次结果。
